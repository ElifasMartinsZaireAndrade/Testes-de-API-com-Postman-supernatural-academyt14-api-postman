API Supernatural Academy

Esta API fornece endpoints para gerenciamento de personagens, episódios e temporadas da série Supernatural. Os testes incluem validações de CRUD, autenticação e integridade dos dados.

  • Testes de endpoints GET, POST, PUT, DELETE
  • Validação de schemas JSON
  • Testes de performance e tempo de resposta
  • Verificação de códigos de status HTTP
